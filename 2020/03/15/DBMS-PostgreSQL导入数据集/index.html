查看数据集信息的命令

我们可以在psql环境中输入：

  • \d;, \d+; 可以查看当前数据库中的表格
  • \d <table_name>;, \d+ <table_name>;: 可以查看对应表格的schema


另外我们也可以在命令行中输入：

  • psql -d testdb -c '\d'
  • psql -d testdb -c '\d+'
  • psql -d testdb -c '\d <table_name>'
  • psql -d testdb -c '\d+ <table_name>'
